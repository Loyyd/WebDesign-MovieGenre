Konrad Kunkel

  • For a long time is was trying to make image sources (src) to use absolute paths, but I realized that absolute paths are not working for local testing since my browser just opens an html file and doesn't know where the defined root is for the site. So instead I converted all paths to use relative paths. This way I ensure pictures are getting loaded properly as long as the file structure is the same.
  • Another mistake I made was to mix up my naming convention for classes. At the beginning of the project I used different naming and didnt realize that the css files are getting messy. So I renamed some classes to keep it more consistent.

Alexandru Fechete

  • At the beginning of the project, I didn’t organize my sections of the website very well, which made it difficult to coordinate with the rest of the team. I worked on certain pages without fully aligning them with our overall layout and design style, which meant I had to redo parts of my work later to ensure consistency.

I didn’t add alt text to some images, and a few color choices didn’t meet proper contrast guidelines, which could make the content harder to read for some users. Because of this, I had to spend extra time going back through my pages, adjusting the colors, adding descriptive alt text, and making sure the layout was usable for all visitors.
